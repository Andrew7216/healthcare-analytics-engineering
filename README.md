# Healthcare Analytics Platform

## Overview

This project demonstrates the design and implementation of a healthcare analytics platform using **dbt Core** and **Snowflake**.

The project follows a layered Analytics Engineering architecture:

- Raw Sources
- Staging
- Intermediate
- Marts
- Reporting

## Features

- Patient, Provider, Visit, Appointment, and Lab data models
- Fact and Dimension modeling
- Diabetes A1C Care Gap
- Lead Screening Care Gap
- Care Gap Outreach Worklist
- Care Gap History
- dbt documentation and testing

## Technology

- dbt Core
- Snowflake
- SQL
- Git
- GitHub
